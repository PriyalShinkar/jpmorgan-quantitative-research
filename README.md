# JPMorgan Chase Quantitative Research Virtual Experience

This repository contains Python-based quantitative research and risk analysis projects completed as part of the JPMorgan Chase & Co. Forage Virtual Experience Program.

---

# Projects Included

## 1. Natural Gas Price Analysis
- Analyzed historical natural gas prices
- Visualized seasonal trends
- Estimated future gas prices using interpolation and forecasting

### Skills Used
- Python
- Pandas
- NumPy
- Matplotlib
- Data Forecasting

---

## 2. Gas Storage Pricing Model
- Built a pricing model for natural gas storage contracts
- Calculated contract value using:
  - Injection dates
  - Withdrawal dates
  - Storage costs
  - Volume limits

### Skills Used
- Financial Modeling
- Commodity Pricing
- Python Functions

---

## 3. Credit Risk Analysis
- Developed a machine learning model to estimate Probability of Default (PD)
- Calculated Expected Loss using recovery rate assumptions

### Skills Used
- Logistic Regression
- Scikit-learn
- Risk Analytics
- Predictive Modeling

---

## 4. FICO Score Bucketing
- Created FICO score buckets for mortgage risk segmentation
- Assigned borrower ratings based on credit score ranges
- Calculated probability of default for each bucket

### Skills Used
- Quantization
- Credit Risk Modeling
- Data Analysis
- Python
